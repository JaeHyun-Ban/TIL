# 2. 프로그램 구조
---
## 2. 변수와 상수
### 1. 주석(comment)
- 소스코드만으로는 이해에 어려움이 있어서 이해를 돕기위해 주석 사용
- C 주석: /*   */, //

### 2. 예약어(keyword)
- 약속된 의미의 단어
- 약속된 의미로만 사용 가능
- 사용자가 재정의해서 사용 수 없음

### 3. 식자(identifier)
- 사용자가 만든 이름(변수명, 함수명)
- 영문대소문자 따로 구별(abc != Abc)
- 영문대소문자, 밑줄(_), 숫자를 혼합하여 생성
- **첫 자는 숫자가 오면 안됨**
	
### 4. 상수(constant)
- 프로그램 내에서 항 고정된 값을 의미
