## 작성 규칙

- ~~폴더와 파일의 이름은 영어로 작성한다.~~

  > 무리해서 작성하지말고 차근차근

- 배운내용에 추가로 알게된 점이 있다면 업데이트를 진행할 것이다

- 검색을통해 알게된것은 외부링크를 첨부해서 작성한다

- 작성한 것은 README에 링크를 생성해 알파벳 순서대로 정렬한다.

- 모든 문서는 마크다운(Markdown, .md)로 작성한다

- **NEW** 작성 방법 변경
