# TIL(Today I Learned)

## 작성 규칙

- 폴더와 파일의 이름은 영어로 작성한다.

  > 가급적이면

- 배운내용에 추가로 알게된 점이 있다면 업데이트를 진행할 것이다

- 검색을통해 알게된것은 외부링크를 첨부해서 작성한다

- 배운것들은 README에 링크를 생성해 정렬한다.

- 모든 문서는 Markdown으로 작성한다

- TIL문서는 최대한 간략하게 작성하려고 하기.
