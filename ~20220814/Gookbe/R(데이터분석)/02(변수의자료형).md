# 변수

다양한 값을 지니고 있는 하나의 속성이며, 변수는 데이터 분석의 대상이다

---

## 변수 생성하기

R에서는 주로 화살표로 변수를 생성한다

> 화살표 단축키: alt + -

```
a <- 1
b <- 20
a <- "hello world"
a <- 3.14

```

---

## 연속 변수 생성하기(중요)

### c(): combine()

숫자 문자를 동시에 저장하면 문자형으로 변경된다

```
var1 <- c(1,2,5,7,8)
> 낟개로 표현
[var1] 1 2 5 7 8

var2 <- c(1:5)
>연속된 변수의 표현
[var2] 1 2 3 4 5

```
