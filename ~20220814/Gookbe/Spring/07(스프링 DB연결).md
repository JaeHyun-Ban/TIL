# Spring - JDBC

## Spring JDBC란(jdbcTemplate)

- JDBC의 장접을 유지하면서, 전통적방식의 JDBC작업을 극복해 간결한 API사용법을 제공
  기존 방식에서 지원하지 않는 편리한 기능을 제공함
- Spring JDBC는 반복적으로 하는 작업을 대신 진행함
