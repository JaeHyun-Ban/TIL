## 데이터 파악하기

### 데이터 프레임 함수

- head(): 데이터 앞부분 출력
- tail(): 데이터 뒷부분 출력
- View(): 뷰어 창에서 데이터 확인
- str(): 데이터 속성 출력
- dim(): 데이터 차원 출력
  > demention(차원)
- nrow(): 데이터 행 출력
- ncol(): 데이터 열 출력
- summary(): 요약 통계량 출력
  > 간단한 정보를 알려준다
- rownames()(★): 행이름
  > 가져온 데이터의 행이름만 추출
- colnames()(★): 열이름
  > 가져온 데이터의 열이름만 추출

## 메모

- 4분위수: `|----|----|----|`
  - 데이터를 4등분해서 나온 위치

3rd Qu. : 4분위 쿼터 중 3번째 위치 `|----|----|====|`
시작부터 끝까지를 4등분으로 나눠 3번째로 나눠진 위치

통계학에서 신뢰할 수 있는 데이터 구간이 4분위 수 라고 함(나머지는 신뢰하기 힘듦)
