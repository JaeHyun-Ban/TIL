# 매개 변수와 리턴

## 매개 변수

- 함수가 필요한 값을 전달하는 매개체
- 자바스크립트의 매개 변수는 함수를 호출하는데 영향을 끼치지 않는다

---

# 함수의 가변인자 argument

자바스크립트에서 매개변수는 큰 의미가 없다
단순히 인자 값에 들어오는 것에 이름을 붙이는 형태이다

```
method(1,2,3,4,5,6,7);

function method(a, b) {
  return a + b;
}
```

위의 코드는 에러를 발생시키지 않고 실행된다
